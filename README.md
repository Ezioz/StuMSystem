## 文件夹说明
1.demo文件夹下是发布的exe文件，在无qt5.8环境也可运行。进入文件夹双击StudentInfoSystem.exe即可。
2.SIMSm文件夹下是工程文件,SIMSm/datastore/user.txt是用户名与密码存储文件。Debug与Release文件夹为Qt自动生成。
SIMSm/uidesign存储ui设计所使用的图片。
